# LenFi Finance SDK

This SDK provides easy integration with LenFi Finance on the Cardano blockchain.

## Installation

```bash
npm install lenfi-finance-sdk
```

## Usage

```typescript
import { LenfiClient } from 'lenfi-finance-sdk';

// Initialize the client
const client = new LenfiClient();

// Use the SDK...
```
